Custom Cargo Registry (GitHub Pages)

This site hosts a sparse Cargo registry index and crate downloads.

Configure Cargo

Create or update .cargo/config.toml:

[registries]
my-registry = { index = "sparse+https://acciente.github.io/custom-rust-registry.github.io/" }

Use the demo crate

[dependencies]
hello-world = { version = "0.1.0", registry = "my-registry" }

The crate exposes hello_world::hello() which returns "hello world".
